# Bag Shop - 프리미엄 가방 쇼핑몰 🛍️

Three.js를 활용한 3D 가방 쇼핑몰 웹사이트입니다. 현대적인 디자인과 인터랙티브한 사용자 경험을 제공합니다.

## ✨ 주요 기능

### 🎯 레이아웃 구성
- **네비게이션 메뉴**: 상단 고정형 메뉴로 부드러운 스크롤 지원
- **MD 추천 상품**: 중앙 쇼케이스 형태의 3D 가방 전시
- **인기상품 목록**: 슬라이더 형태의 상품 목록

### 🎨 MD 추천 상품 기능
- **Three.js 3D 모델링**: 실시간 렌더링된 3D 가방 모델
- **색상 변경**: 4가지 색상 옵션 (갈색, 검정, 파란, 회색)
- **마우스 인터랙션**: 마우스 움직임에 따른 가방 회전
- **자동 회전**: 3D 모델의 자동 회전 애니메이션

### 📦 인기상품 목록 기능
- **슬라이더**: 6개 상품을 한 번에 표시
- **네비게이션**: 좌우 버튼으로 상품 탐색
- **더보기 기능**: 추가 상품 동적 로드
- **호버 효과**: 상품 카드 마우스 오버 시 애니메이션

### 📱 반응형 디자인
- **데스크톱**: 최적화된 레이아웃 (1200px 이상)
- **태블릿**: 중간 화면 최적화 (768px - 1199px)
- **모바일**: 작은 화면 최적화 (767px 이하)

## 🚀 시작하기

### 사전 요구사항
- 최신 웹 브라우저 (Chrome, Firefox, Safari, Edge)
- 로컬 웹 서버 (Python, Node.js 등)

### 설치 및 실행

1. **저장소 복제**
```bash
git clone [repository-url]
cd bagshop
```

2. **로컬 서버 실행**
```bash
# Python 3 사용 시
python3 -m http.server 8000

# Node.js 사용 시
npx http-server -p 8000

# PHP 사용 시
php -S localhost:8000
```

3. **웹사이트 접속**
브라우저에서 `http://localhost:8000` 접속

## 🎮 사용자 인터랙션

### 3D 가방 조작
1. **자동 회전**: 가방이 자동으로 천천히 회전
2. **마우스 인터랙션**: 마우스를 움직여 가방 각도 조절
3. **색상 변경**: 색상 버튼 클릭으로 실시간 색상 변경

### 상품 슬라이더
1. **네비게이션**: 좌우 화살표 버튼으로 상품 이동
2. **더보기**: 추가 상품 로드
3. **호버**: 상품 카드에 마우스 오버 시 확대 효과

## 🔧 커스터마이징

### 색상 테마 변경
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --text-color: #2c3e50;
    --background-color: #f8f9fa;
}
```

### 3D 모델 수정
```javascript
// script.js의 createBagModel() 함수에서 조절
const bagGeometry = new THREE.BoxGeometry(2, 2.5, 1); // 크기 조절
const bagMaterial = new THREE.MeshPhongMaterial({ 
    color: currentColor,
    shininess: 100  // 광택 조절
});
```
